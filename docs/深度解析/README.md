## 文档结构

每个文档都包含：
- 概念解释（为什么需要？）
- 数据结构详解（位字段图）
- 代码逐行分析
- 流程图
- 调试技巧
- 总结速查表

## 学习建议

1. **先看概念**：理解"为什么"比"怎么做"更重要
2. **画图理解**：特别是栈变化和地址转换
3. **对照代码**：边看文档边读 `student-distrib/` 下的源码
4. **动手调试**：用 GDB 单步执行，验证理解
